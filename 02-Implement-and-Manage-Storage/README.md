## 🔑 Key Sub-Topics

### 2.1 Secure Storage
- Configure storage firewalls and virtual networks
- Configure storage account access keys
- Configure shared access signatures (SAS)
- Configure stored access policies
- Configure Microsoft Entra ID authorization
- Configure storage encryption
- Configure private endpoints

### 2.2 Manage Storage
- Create and configure storage accounts
- Configure redundancy and replication (LRS, ZRS, GRS, GZRS)
- Configure object replication
- Configure storage access tiers (Hot, Cool, Cold, Archive)
- Import and export data (AzCopy, Data Box, Import/Export)
- Configure soft delete and versioning

### 2.3 Configure Azure Files and Azure Blob Storage
- Configure Azure Blob Storage (containers, blobs, access levels)
- Configure Azure Files (file shares, tiers, quotas)
- Mount Azure file shares
- Configure Azure File Sync
- Configure blob snapshots and leases

---

> **Key exam areas**: SAS types (User Delegation, Service, Account), access tiers, redundancy options, Azure File Sync, and lifecycle management policies.
